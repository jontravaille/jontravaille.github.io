Falling Knives

"10/10 Recommend!"

"Frightening, but Unstoppable"




What is This Game About

Fate has given you, a game player, to avoid countless and terrifying knives falling from the sky. There is no way to escape from this -- survive as long as you can!

How to Play

There's only two keys you should remember: left arrow key and right arrow key.

How to Make Things Easier / Harder

If you think it is quite easy enough to live long, press "1" and "3". Let's see whether things are still the same! On the other hand, if you keep dying so much that you be frustrated badly and can't move forward, press "2"and "4". Hope things could be much easier.
